HTML for Absolute Beginners — Start Here

If you have never written a single line of code in your life, this article is for you. By the end of this guide, you will understand what HTML is, how it works, and you will have built your very first webpage.

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. Every website you have ever visited — Google, YouTube, Instagram — all of them use HTML as their foundation.

Think of HTML like the skeleton of a webpage. It gives structure to your content. CSS adds the style (colors, fonts), and JavaScript adds behavior (clicks, animations). But everything starts with HTML.

How Does HTML Work?

HTML uses something called tags to tell the browser what kind of content to display. A tag looks like this:

<p>This is a paragraph.</p>

Notice two things:

  1. The tag opens with <p>
  2. The tag closes with </p> (notice the forward slash)

Everything between the opening and closing tag is the content that will appear on the page.

💡 Tip: Most HTML tags come in pairs — an opening tag and a closing tag. Always remember to close your tags!

Your First HTML Page

Every HTML page follows the same basic structure. Here is the simplest possible webpage:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

What Does Each Part Mean?

  • <!DOCTYPE html> — Tells the browser this is an HTML5 document
  • <html> — The root element. Everything goes inside here
  • <head> — Contains information about the page (not visible to users)
  • <title> — The text that appears in the browser tab
  • <body> — Everything the user sees goes here
  • <h1> — A heading (h1 is the biggest, h6 is the smallest)
  • <p> — A paragraph of text

Try It Yourself!

Here is how to create your first webpage right now:

  1. Open Notepad (Windows) or TextEdit (Mac)
  2. Copy the code above and paste it
  3. Save the file as index.html
  4. Open the file in your browser

You should see "Hello World!" displayed as a big heading on the page. Congratulations — you just wrote your first HTML!

🎉 You did it! Every professional web developer started exactly where you are right now. Keep practicing and you will be building full websites before you know it.
