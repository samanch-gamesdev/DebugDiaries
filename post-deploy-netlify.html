How to Deploy Your Website on Netlify for Free

S
By Saman·6 min read·July 2026

You have built your first website — congratulations! Now it is time to share it with the world. Netlify is one of the easiest and most popular ways to host a website completely for free. In this guide, you will learn exactly how to deploy your website on Netlify step by step.

By the end of this tutorial, your website will be live on the internet with a real URL that anyone can visit!

What is Netlify?

Netlify is a free web hosting platform that makes it incredibly easy to deploy websites. It is especially popular among front-end developers because it works perfectly with HTML, CSS, and JavaScript websites. Some of the best features of Netlify include:

  • Free hosting with a custom netlify.app subdomain
  • Automatic deployments when you push to GitHub
  • Free SSL certificate (HTTPS) for all sites
  • Fast global content delivery network (CDN)
  • Easy drag and drop deployment

Method 1: Drag and Drop (Easiest)

The fastest way to deploy on Netlify is to simply drag and drop your project folder onto the Netlify website. Here is how:

1

Create a Netlify Account

Go to netlify.com and sign up for a free account. You can sign up with your GitHub account for even easier setup.

2

Go to Your Dashboard

After logging in, you will see your Netlify dashboard. Scroll down and you will see a drag and drop area.

3

Drag Your Project Folder

Open your file explorer, find your project folder, and drag it directly into the Netlify drop zone. Make sure your main file is named index.html!

4

Your Site is Live!

Netlify will automatically deploy your site and give you a random URL like amazing-site-123.netlify.app. Your site is now live!

Method 2: Deploy from GitHub (Recommended)

The better way to use Netlify is to connect it to your GitHub repository. This way, every time you push an update to GitHub, Netlify automatically updates your live site too!

1

Push Your Code to GitHub

Make sure your project is on GitHub. Create a repository and upload all your files.

2

Click "Add New Site" in Netlify

In your Netlify dashboard, click "Add new site" then "Import an existing project".

3

Connect GitHub

Click "GitHub" and authorize Netlify to access your repositories. Then select the repository you want to deploy.

4

Deploy!

Click "Deploy site" and Netlify will build and deploy your site automatically. From now on, every GitHub push will trigger a new deployment!

🎉 Pro Tip: Make sure your main HTML file is always named index.html. Netlify (and all web servers) automatically serve index.html as the homepage. If your file has any other name, visitors will get a "Page not found" error.

Changing Your Site Name

By default, Netlify gives your site a random name. You can change it to something more memorable:

  1. Go to your site dashboard in Netlify
  2. Click "Site configuration" or "Site settings"
  3. Under "Site details", click "Change site name"
  4. Enter your preferred name (e.g., myportfolio.netlify.app)
