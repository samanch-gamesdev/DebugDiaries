How to Build a Navigation Bar with HTML and CSS

S
By Saman·7 min read·July 2026

A navigation bar is one of the most important parts of any website. It helps users move between pages and find what they are looking for. In this tutorial, you will learn how to build a clean, professional navigation bar from scratch using only HTML and CSS — no JavaScript needed!

By the end of this guide, you will have a fully working navigation bar that you can add to any website project.

What is a Navigation Bar?

A navigation bar (also called a navbar) is a section of a website, usually at the top, that contains links to the main pages of the site. You see navigation bars on every website you visit — they usually have the logo on the left and links on the right.

A good navigation bar should be easy to use, always visible, and consistent across all pages of the website. It is usually the first thing a visitor sees when they land on your site, so it is important to get it right.

The HTML Structure

Let us start with the HTML. We will use the semantic HTML5 nav element, which is the correct way to mark up a navigation section:

<nav class="navbar">
  <a href="index.html" class="logo">MyWebsite</a>
  <ul class="nav-links">
    <li><a href="index.html">Home</a></li>
    <li><a href="about.html">About</a></li>
    <li><a href="blog.html">Blog</a></li>
    <li><a href="contact.html">Contact</a></li>
  </ul>
</nav>

Notice we are using a ul (unordered list) for the navigation links. This is the correct and semantic way to build a nav menu — it tells screen readers and search engines that these are a list of related links.

Basic CSS Styling

Now let us add CSS to make the navigation bar look good. We will use Flexbox to align the logo and links:

.navbar {
  background-color: #1A2E29;
  padding: 0 2rem;
  display: flex;
  align-items: center;
  justify-content: space-between;
  height: 64px;
}

.logo {
  color: white;
  text-decoration: none;
  font-size: 1.3rem;
  font-weight: 700;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 2rem;
}

.nav-links a {
  color: rgba(255, 255, 255, 0.8);
  text-decoration: none;
  font-size: 0.95rem;
  transition: color 0.2s;
}

.nav-links a:hover {
  color: white;
}

Making the Navbar Sticky

A sticky navbar stays at the top of the screen even when the user scrolls down. This is very important for user experience because users should always be able to navigate without scrolling back to the top. Here is how to make your navbar sticky:

.navbar {
  position: sticky;
  top: 0;
  z-index: 100;
  /* rest of your styles */
}
💡 Important: The z-index: 100 ensures the navbar stays on top of all other content on the page. Without this, other elements might appear over the navbar when you scroll.

Adding a Hover Effect

Let us add a nice underline effect when the user hovers over a link. This gives great visual feedback:

.nav-links a {
  color: rgba(255, 255, 255, 0.8);
  text-decoration: none;
  padding-bottom: 4px;
  border-bottom: 2px solid transparent;
  transition: all 0.2s;
}

.nav-links a:hover {
  color: white;
  border-bottom-color: #2DA58E;
}

The Complete Code

Here is the complete HTML and CSS for your navigation bar all in one place:

<!-- HTML -->
<nav class="navbar">
  <a href="index.html" class="logo">MyWebsite</a>
  <ul class="nav-links">
    <li><a href="index.html">Home</a></li>
    <li><a href="about.html">About</a></li>
    <li><a href="blog.html">Blog</a></li>
  </ul>
</nav>

<!-- CSS -->
<style>
.navbar { background:#1A2E29; padding:0 2rem; display:flex;
  align-items:center; justify-content:space-between; height:64px;
  position:sticky; top:0; z-index:100; }
.logo { color:white; text-decoration:none; font-size:1.3rem; font-weight:700; }
.nav-links { list-style:none; display:flex; gap:2rem; }
.nav-links a { color:rgba(255,255,255,0.8); text-decoration:none; transition:color 0.2s; }
.nav-links a:hover { color:white; }
</style>
