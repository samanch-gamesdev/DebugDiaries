Git & GitHub: Your First Repository in 10 Minutes

S
By Saman· 5 min read· June 2026

Every developer needs to know Git. It is a tool that saves your code history and lets you share your work online. In this guide, you will create your first GitHub repository in just 10 minutes — no experience needed!

Advertisement

What is Git?

Git is a version control system. Think of it like a save button for your entire project — but smarter. It remembers every change you made, so you can always go back to an older version if something breaks.

What is GitHub?

GitHub is a website where you store your Git projects online. It is like Google Drive but specifically for code. Millions of developers use it to share and collaborate on projects.

💡 Simple Analogy: Git is the tool on your computer. GitHub is the online storage. Git saves your work locally, GitHub saves it online for the world to see.

Step by Step: Create Your First Repo

1

Create a GitHub Account

Go to github.com and sign up for a free account. Choose a professional username — this will be part of your portfolio!

2

Create a New Repository

Click the green "New" button on your dashboard. Give your repo a name, set it to Public, and click "Create repository".

3

Upload Your Files

Click "uploading an existing file", drag and drop your HTML/CSS files, then click "Commit changes".

4

Your Code is Now Online!

Visit github.com/yourusername/yourrepo — your code is now live on the internet for anyone to see!

Basic Git Commands

If you want to use Git from the command line, here are the most important commands:

git init # Start a new git project
git add . # Add all files to staging
git commit -m "msg" # Save your changes
git push # Upload to GitHub

Why Every Developer Needs GitHub

  • It acts as your portfolio — employers check GitHub
  • You never lose your code — it is backed up online
  • You can collaborate with other developers
  • You can deploy websites for free using GitHub Pages or Netlify
🎉 You did it! Having a GitHub profile with projects is one of the best things a CS student can do for their career. Start uploading your projects today!

What to Learn Next?

  • GitHub Pages — host your website for free
  • Netlify — even easier free hosting
  • Git branches — work on features without breaking your main code
