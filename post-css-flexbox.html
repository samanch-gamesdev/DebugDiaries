CSS Flexbox Explained — With Real Examples

S
By Saman· 4 min read· June 2026

Flexbox is one of the most useful CSS tools for beginners. Once you understand it, you will be able to create layouts that previously seemed impossible. Let us break it down simply.

What is Flexbox?

Flexbox (Flexible Box Layout) is a CSS layout system that makes it easy to arrange elements in a row or column. It helps you align and distribute space among items in a container.

Before Flexbox, aligning things in CSS was very hard. With Flexbox, it becomes simple!

How to Enable Flexbox

To use Flexbox, add display: flex to the parent container:

.container {
  display: flex;
}

That is it! Now all the children inside become flex items.

flex-direction

This property controls whether items go in a row or column.

.container {
  display: flex;
  flex-direction: row; /* default */
  /* or: flex-direction: column; */

justify-content

This controls how items are positioned horizontally (along the main axis).

.container {
  display: flex;
  justify-content: center; /* center items */
  /* other values: flex-start, flex-end, space-between, space-around */
}

align-items

This controls how items are positioned vertically (along the cross axis).

.container {
  display: flex;
  align-items: center; /* vertically center */
}
💡 Quick Trick: To center something both horizontally and vertically, use:

display: flex; justify-content: center; align-items: center;

This is the most common flexbox pattern every developer uses!
