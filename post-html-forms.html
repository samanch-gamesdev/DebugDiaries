HTML Forms Tutorial for Beginners — Complete Guide

S
By Saman·8 min read·July 2026

Forms are everywhere on the internet — login pages, contact forms, search bars, signup pages, and much more. Learning how to create HTML forms is an essential skill for every web developer. In this tutorial, you will learn everything you need to know about HTML forms from scratch.

Advertisement

What is an HTML Form?

An HTML form is a section of a webpage that allows users to enter and submit data. When a user fills out a form and clicks submit, the data is sent to a server for processing. Forms are used for everything from logging in to a website to placing an online order.

The Basic Form Structure

Every HTML form starts with the form element. Here is the most basic form possible:

<form action="/submit" method="POST">
  <!-- form elements go here -->
</form>

The form element has two important attributes:

  • action — the URL where the form data will be sent when submitted
  • method — how the data is sent (GET or POST)

Input Fields

The input element is the most common form element. It creates different types of input fields depending on the type attribute:

<!-- Text input -->
<input type="text" placeholder="Enter your name"/>

<!-- Email input -->
<input type="email" placeholder="Enter your email"/>

<!-- Password input -->
<input type="password" placeholder="Enter password"/>

<!-- Number input -->
<input type="number" placeholder="Enter a number"/>

<!-- Submit button -->
<input type="submit" value="Submit Form"/>

Labels — Always Use Them!

Every input field should have a label that describes what it is for. Labels are important for accessibility and usability:

<label for="username">Username:</label>
<input type="text" id="username" name="username"/>
💡 Tip: The for attribute on the label should match the id attribute on the input. This connects them so clicking the label also focuses the input field — which is great for usability!

A Complete Contact Form

Here is a complete, styled contact form with all the common elements:

The Code for This Form

<form>
  <label for="name">Full Name</label>
  <input type="text" id="name" placeholder="Enter your name"/>

  <label for="email">Email Address</label>
  <input type="email" id="email" placeholder="your@email.com"/>

  <label for="message">Message</label>
  <textarea id="message" rows="4"></textarea>

  <button type="submit">Send Message</button>
</form>
