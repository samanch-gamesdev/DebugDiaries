CSS Grid Explained for Beginners — With Real Examples

S
By Saman·8 min read·July 2026

If you have ever tried to create a webpage layout with rows and columns, CSS Grid is exactly what you need. It is one of the most powerful layout tools in CSS, and once you learn it, you will wonder how you ever built websites without it.

In this guide, we will start from the very basics and build up to real examples you can use in your own projects. By the end, you will understand how CSS Grid works and feel confident using it.

What is CSS Grid?

CSS Grid is a two-dimensional layout system for the web. Unlike Flexbox, which works in one direction (either rows or columns), CSS Grid lets you work with both rows and columns at the same time. This makes it perfect for creating page layouts, card grids, and complex designs.

Think of CSS Grid like a table in Microsoft Word or Excel. You define how many rows and columns you want, and then you place your content inside the cells. The big difference is that CSS Grid is much more flexible and powerful than a regular table.

Why Learn CSS Grid?

Before CSS Grid, developers had to use complicated hacks with floats, positioning, and tables to create layouts. CSS Grid makes all of this much easier. Here are some reasons why every web developer should learn CSS Grid:

  • It makes complex layouts simple and easy to understand
  • It works in two dimensions — rows AND columns simultaneously
  • It reduces the amount of code you need to write
  • It is supported by all modern browsers
  • It combines perfectly with Flexbox for even more control

How to Enable CSS Grid

Just like Flexbox, enabling CSS Grid is very simple. You just need to add one line of CSS to your container element:

.container {
  display: grid;
}

That is it! Now the element is a grid container and all its children become grid items. But just enabling grid does not do much yet — we need to define our rows and columns.

Defining Columns with grid-template-columns

The most important property in CSS Grid is grid-template-columns. This is where you define how many columns you want and how wide each one should be.

.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
}

This creates three columns, each 200 pixels wide. But typing the same value three times is repetitive. CSS Grid has a special function called repeat() that makes this easier:

.container {
  display: grid;
  grid-template-columns: repeat(3, 200px);
  /* Same as: 200px 200px 200px */
}

The fr Unit — Flexible Fractions

One of the most useful features of CSS Grid is the fr unit. It stands for "fraction" and it lets you divide the available space proportionally. This is much better than using pixels because it automatically adjusts to any screen size.

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  /* Three equal columns */
}

You can also mix fr units with fixed sizes. For example, you can create a layout with a fixed sidebar and a flexible main content area:

.container {
  display: grid;
  grid-template-columns: 250px 1fr;
  /* Fixed sidebar + flexible content */
}

Adding Gaps Between Items

By default, grid items sit right next to each other with no space in between. You can add spacing using the gap property:

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px; /* Space between all items */
}

You can also control the row gap and column gap separately using row-gap and column-gap.

Defining Rows with grid-template-rows

Just like you can define columns, you can also define the height of your rows using grid-template-rows:

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 200px;
  /* First row 100px, second row 200px */
}

Grid vs Flexbox — Which One to Use?

A very common question for beginners is: when should I use Grid and when should I use Flexbox? Here is a simple rule to remember:

  • Use Flexbox for one-dimensional layouts — arranging items in a single row or column, like a navigation bar or a row of buttons
  • Use Grid for two-dimensional layouts — when you need to control both rows and columns, like a page layout or a card grid
💡 Pro Tip: You do not have to choose between Grid and Flexbox! The best websites use both together. Use Grid for the overall page layout, and use Flexbox for aligning items inside each grid area.

A Real Example: Blog Card Grid

Here is a practical example of CSS Grid — creating a responsive grid of blog cards:

.cards-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
  padding: 2rem;
}

.card {
  background: white;
  border-radius: 12px;
  padding: 1.5rem;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

What to Learn Next?

Now that you understand the basics of CSS Grid, here is what to explore next:

  • grid-column and grid-row — make items span multiple columns or rows
  • grid-template-areas — name your grid areas for easier placement
  • auto-fit and auto-fill — create truly responsive grids without media queries
  • minmax() — set minimum and maximum sizes for grid tracks
