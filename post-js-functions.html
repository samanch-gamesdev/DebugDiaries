JavaScript Functions for Beginners — Complete Guide

S
By Saman·9 min read·July 2026

Functions are one of the most important concepts in JavaScript — and in programming in general. Once you understand functions, you will be able to write much better, cleaner, and more organized code. In this guide, we will cover everything you need to know about JavaScript functions as a beginner.

What is a Function?

A function is a reusable block of code that performs a specific task. Instead of writing the same code over and over again, you write it once inside a function and then call that function whenever you need it.

Think of a function like a recipe. You write the recipe once, and then any time you want to make that dish, you just follow the recipe. You do not write it out again from scratch every time.

Why Use Functions?

Functions are essential in programming for several reasons:

  • Reusability — Write code once, use it many times
  • Organization — Break your code into small, manageable pieces
  • Readability — Well-named functions make code easy to understand
  • Debugging — Easier to find and fix bugs in small functions
  • Maintainability — If you need to change something, you only change it in one place

How to Write a Function

In JavaScript, the most common way to write a function is using the function keyword:

function greet() {
  console.log("Hello! Welcome to DebugDiaries!");
}

// Call the function
greet(); // Output: Hello! Welcome to DebugDiaries!

Let us break this down:

  • function — the keyword that tells JavaScript we are creating a function
  • greet — the name we give our function
  • () — parentheses where we can pass information into the function
  • {} — curly braces that contain the code the function will run

Functions with Parameters

Parameters allow you to pass information into a function so it can work with different data each time it is called:

function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Saman"); // Output: Hello, Saman!
greet("Ahmed"); // Output: Hello, Ahmed!
greet("Fatima"); // Output: Hello, Fatima!

Now the same function works for any name! You can also have multiple parameters:

function add(num1, num2) {
  console.log(num1 + num2);
}

add(5, 3); // Output: 8
add(10, 20); // Output: 30

Functions that Return Values

Sometimes you want a function to calculate something and give you back the result. You use the return keyword for this:

function multiply(num1, num2) {
  return num1 * num2;
}

const result = multiply(4, 5);
console.log(result); // Output: 20

// You can also use the return value directly
console.log(multiply(3, 7)); // Output: 21
💡 Important: When a function hits a return statement, it stops running immediately and sends back the value. Any code after return inside the function will not run.

Arrow Functions — The Modern Way

In modern JavaScript (ES6 and later), there is a shorter way to write functions called arrow functions. They use an arrow (=>) instead of the function keyword:

// Regular function
function greet(name) {
  return "Hello, " + name;
}

// Arrow function — same thing, shorter syntax
const greet = (name) => {
  return "Hello, " + name;
}

// Even shorter — when there is only one line
const greet = (name) => "Hello, " + name;

Arrow functions are very popular in modern JavaScript code. You will see them everywhere once you start reading other developers code.

Default Parameters

You can give your parameters default values that are used when no value is passed in:

function greet(name = "friend") {
  return "Hello, " + name + "!";
}

console.log(greet("Saman")); // Hello, Saman!
console.log(greet()); // Hello, friend!

  • Callback functions — passing functions as arguments to other functions
  • Higher-order functions — map, filter, reduce
  • Async functions — handling asynchronous operations
